## About
Data extraction and transformation from DTO to be used in PH tools.

* data_catalog: JSON metadata files for each PH describing ready to use dataset. 
* data_flows: scripts to extract and transform data. 
* data_sets: datafiles to be used in PH applications.



## Notice
The data\_catalog json metadata files are used by PH-applications to dynamically import datasets. 
More info te be provided.
